Playground for declarative vitest function mock.

## Modules

- Declare expected calls as a list of args/returnValue pairs: [module](src/mock-fn-calls.ts), [spec](src/mock-fn-calls.spec.ts)

- Record-and-Replay function calls using snapshots and serialization: [module](src/mock-fn-record-and-replay.ts), [spec](src/mock-fn-record-and-replay.spec.ts)

## Setup

```sh
$ nvm use
```
